
# Subarray

## Algorithm

* DP
* Sliding Window
* Prefix Sum
* Mono Queue
* Mono Stack

## Types

1. Sum of subarray:

    max sum subarray of any size

2. Count of subarray:

    subarray sum == k

3. Size of subarray:

    min size subarray sum >= k(positive)
    min size subarray sum >= k(positive and negative)

4. Count of subarray with k difference elements

5. Count of subarray with at most k difference elements

6. Longest min-max difference <= k
